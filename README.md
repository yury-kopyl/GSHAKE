# GSHAKE
1. Install [Node.js](https://nodejs.org/en/download/)
2. Install [Sass](http://sass-lang.com/install)
3. Clone/copy files in project
4. Open cmd and cd project folder
5. Enter `npm install` in cmd
6. Enjoy!!! :boom:
